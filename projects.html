React Native Artist Albums
 Github Repo

I developed a better understanding of React Native while creating this project that lists albums from an artist and directs the user to amazon website if they click on the `Buy` button. The album list is retrieved using an `HTTP GET` request done by axios library and takes advantage of the React Component lifecycle to do it.

React Light Dashboard App
 Github Repo  Deployed Application

I developed a better understanding of React - state, lifecycle, deployment - while creating this project that provides a dashboard to control lights in a web interface. By interacting with a rest api the application retrieves the set of lights and updates each light brightness according to a custom HTML 5 slider that represents a light intensity. Some technologies used in this app are: React, ESLint, Prettier, HTML 5, Firebase Hosting.
